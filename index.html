SpeedX - Online Speedometer in your pocket

SpeedX helps you find your real-time speed in km/h. No installation needed. Use it right away! Speedify is also available as progressive web app which you can use offline in latest browsers like Google Chrome.

If you find SpeedX useful, please Soumyadeep Dey On Github.

Some browsers and devices have support issues with geoLocation API. If you are not getting correct location, make sure you are in GPS mode in location settings and you gave location permission to both the Browser and Webpage. Or, try with another browser. :)
