SpeedTrackify - Online Speedometer In Your Pocket.

SpeedTrackify helps you find your real-time speed in km/h. No installation needed. Use it right away! SpeedTrackify is also available as a progressive web app which you can use offline in the latest browsers like Google Chrome.

If you find SpeedTrackify useful, please follow Soumyadeep Dey on GitHub.

Note: Some browsers and devices have support issues with the Geolocation API. If you are not getting the correct location, make sure you are in GPS mode in location settings and you have granted location permission to both the Browser and Webpage. Or, try with another browser.
